Reflection

Hello, this was the first time I had ever worked with HTML or created a website in general. As a new learner, I would say the most difficult thing for me was understanding how three different platforms were connected to each other, like changing things in your oxygen shows up on the website and the files saved on the computer, as well as constantly remembering to git push to really save all the changes. Another thing that was difficult to learn is how crucial it is to have absolute knowledge of where everything is located on your local computer. I think that the Penn State site was much easier to create due to the fact that it is much advanced and creates the code for you, but I would prefer GitHub because its advantage is practice. When you do the code by yourself you hold much more power and pride in what you have created on your own. One challenge on GitHub was that I do not know the code language when I wanted to write for example a paragraph I wasn’t sure what kind of code to use to make it a paragraph. There weren’t many challenges with the Penn State website because it was very much self-explanatory. One skill that you absolutely won’t be able to create a website is patience because one little problem could get you stuck in one spot for days and to find it you might have to go all the way back to where you started.
